本專案集錄了我於學期間針對網頁前端架構（HTML/CSS）以及 JavaScript 的實作作品。

### 1. 🌐 Html:Css Layout
* **核心技術：** HTML5 語意化標籤、CSS3 響應式佈局 (Responsive Design)、Flexbox 與 Grid 網格系統。
* **展現能力：** 網頁空間感與視覺轉譯能力，能將設計稿轉化為具備高可讀性與RWD的前端結構。

### 2. ⚡ JavaScript Lab
內含兩大核心實作工具，著重於資料流處理、演算法與防錯機制：

* **表單即時數據驗證與防錯工具 (`tasks.html`)**
  * 實作即時 `input` 事件監聽與動態狀態機切換（綠/橘/紅三色警告提示）。
  * 包含後台「資料去噪機制」，自動過濾電話號碼中的特殊符號（如橫線、點），並在 `submit` 階段實作安全攔截，防止無效資料送出。
* **氣泡排序演算法與數據流治理 (`sorting.html`)**
  * 親手實作經典 **氣泡排序法 (Bubble Sort) 演算法**，透過雙重迴圈（Nested Loops）與臨時變數迭代，自動將亂序數據流進行重組與遞增排序。
  * 內建嚴格的輸入防錯防禦，利用計數器逆向控制機制（`i--`）在第一線阻擋非數字（NaN）與空值。
